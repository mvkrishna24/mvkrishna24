Final-year B.Tech (AI & ML) student graduating in 2026, focused on building **production-grade backends** with Java + Spring Boot and **AI-driven automation** with n8n and LLMs. I care about the parts most demos skip: caching, rate limiting, idempotency, retries, and clean API design. ### Featured Projects

#### [MetroNexis](https://github.com/mvkrishna24/MetroNexis) &nbsp;·&nbsp; AI-Powered Metro Navigation
Converts plain landmarks into optimal metro routes using open geospatial data and Dijkstra-based routing. Built on Spring Boot + PostgreSQL with a React/Vite/Tailwind frontend. **Provisional patent filed with the Indian Patent Office.**
`Spring Boot` · `PostgreSQL` · `Dijkstra` · `React` · `Vite`

#### [url-shortener](https://github.com/mvkrishna24/url-shortener) &nbsp;·&nbsp; Distributed URL Shortener
Production-grade shortener with Base62 ID generation, Redis read-through caching, request rate limiting, async click analytics, and JWT auth — containerized with Docker and tested in CI.
`Java 17` · `Spring Boot` · `Redis` · `PostgreSQL` · `Docker`
&nbsp;→ <a href="#">Live Demo</a> &nbsp;·&nbsp; <a href="https://github.com/mvkrishna24/url-shortener">Source</a>

#### [LeadOps AI CRM](https://github.com/mvkrishna24/leadops-ai-crm) &nbsp;·&nbsp; AI Automation CRM
Lead-management automation that scores incoming leads, sends auto-reply emails, syncs to Google Sheets, and pushes hot-lead alerts — orchestrated in n8n with Gemini/OpenAI.
`n8n` · `Gemini / OpenAI` · `Google Sheets` · `Automation`

#### [notification-service](https://github.com/mvkrishna24/notification-service) &nbsp;·&nbsp; Event-Driven Notifications
Event-driven delivery system with Kafka topics, Redis-backed state, real-time WebSocket push, exponential-backoff retries, and idempotent delivery to prevent duplicates.
`Spring Boot` · `Kafka` · `Redis` · `WebSockets`

#### [GitHub Profile Card Generator](https://github.com/mvkrishna24/github-profile-card-generator) &nbsp;·&nbsp; Web Tool
Generates clean, shareable GitHub profile cards from any username — built with Vanilla JavaScript and the GitHub API.
`Vanilla JS` · `GitHub API`
